# unique-hw-id
A module to generate unique id based on the hardware of the computer.

## Quick Start
Install the module:
```bash
npm i unique-hw-id
```

Generate a unique id:
```js
const { generateID } = require('unique-hw-id');

generateID();
// Output: 74c23ffbd9574163d3c7742ced89084635d6424e630a2a157e0087f4f5ee198c
```
